# Budget Categories

This page helps you choose the correct **Budget Line** (expense account code) when entering expenses in Zoho. If no code is a clear fit, confirm with the Financial Officer before using `899`.


## Compensation & Staff Support

| Code | Account Name | Usage |
|------|--------------|-------|
| `700` | `Employee Salaries (FT)` | Full-time employee salary costs. |
| `701` | `PT Labour - Commercial Events` | Part-time labour directly supporting commercial events. |
| `702` | `PT Labour - AO Subsidies` | Part-time labour for AO subsidy work. |
| `704` | `PT Labour - Grad Events` | Part-time labour for graduate events. |
| `705` | `Employee Benefits (FT)` | Benefits and related employer costs for full-time employees. |
| `706` | `Employee Benefits (PT)` | Benefits and related employer costs for part-time employees. |
| `710` | `Executive Honoraria` | Executive compensation paid as honoraria. |
| `715` | `Honoraria` | Approved honoraria that are not executive honoraria. |
| `730` | `Professional Development - Staff` | Training and professional development for staff. |
| `731` | `Professional Development - Executives` | Training and professional development for executives. |
| `740` | `Staff Appreciation` | Staff appreciation and recognition costs. |
| `880` | `PT Staff Costs - Projects` | Part-time staffing for project delivery. |
| `881` | `PT Staff Costs  - Events` | Part-time staffing for event delivery. |
| `883` | `PT Staff Costs - Orientation` | Part-time staffing for orientation activities. |

## Professional & Contracted Services

## Other / Residual

| Code | Account Name | Usage |
|------|--------------|-------|
| `899` | `Misc.` | Residual or miscellaneous expenses only when no specific budget line applies. |

{: .note }
> If an expense could fit a specific code, use that specific code instead of `899`.
